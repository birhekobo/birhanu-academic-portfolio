# Python for Remote Sensing — Lab Manual

## Overview

This laboratory manual accompanies the Remote Sensing and GIS courses at Wollo University. It provides hands-on exercises using Python for satellite image analysis.

## Modules

1. **Introduction to Python for geospatial data**
   - NumPy, Matplotlib, rasterio basics
   - Reading and visualizing satellite imagery

2. **Spectral indices**
   - NDVI, NDWI, NBR computation
   - Thresholding and classification

3. **Supervised classification**
   - Training data collection
   - Random forest classifier
   - Accuracy assessment

4. **Change detection**
   - Image differencing
   - Post-classification comparison

5. **Time series analysis**
   - MODIS NDVI time series
   - Trend analysis and phenology

## Requirements

- Python 3.10+
- rasterio, geopandas, numpy, matplotlib, scikit-learn
- Google Earth Engine Python API (optional)
