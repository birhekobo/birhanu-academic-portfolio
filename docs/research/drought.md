# Drought

## Overview

Developing satellite-based drought monitoring and early warning systems to support agricultural resilience in Ethiopia, a region highly vulnerable to recurrent drought.

## Drought indices

| Index | Description | Source |
|-------|-------------|--------|
| VCI | Vegetation Condition Index | AVHRR, MODIS |
| TCI | Temperature Condition Index | MODIS |
| VHI | Vegetation Health Index | VCI + TCI |
| SPI | Standardized Precipitation Index | CHIRPS |
| SPEI | Standardized Precipitation-Evapotranspiration Index | ERA5 + CHIRPS |
| NDVI anomaly | Deviation from long-term mean | Landsat, MODIS |

## Methods

- Multi-index drought assessment framework
- Time series analysis of vegetation health
- Machine learning for drought prediction
- Integration with crop yield models

## Applications

- Seasonal drought monitoring bulletins
- Early warning for agricultural agencies
- Historical drought characterization (1980–present)
- Drought risk mapping
