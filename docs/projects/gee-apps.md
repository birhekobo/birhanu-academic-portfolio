# Google Earth Engine Apps

## Overview

Interactive web applications built with Google Earth Engine for environmental monitoring and analysis.

## Apps

### Land Cover Mapper

Interactive tool for supervised land cover classification using Landsat and Sentinel-2 imagery.

- **Training data** — draw polygons on the map
- **Classifier** — random forest, CART, SVM
- **Output** — classified map with accuracy metrics

### NDVI Time Series Explorer

Visualize vegetation index time series for any location.

- **Data** — MODIS, Landsat, Sentinel-2
- **Output** — time series plot with trend line
- **Export** — CSV download

### Change Detection Tool

Bitemporal change detection using Landsat imagery.

- **Method** — image differencing, post-classification comparison
- **Output** — change map with area statistics
