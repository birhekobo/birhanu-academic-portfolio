# Climate Dashboard

## Overview

An interactive web dashboard for visualizing and analyzing climate data, built with Python and modern web technologies.

## Objectives

- Provide intuitive visualization of climate time series
- Enable trend analysis and anomaly detection
- Support multiple data sources (CHIRPS, ERA5, MODIS)
- Generate downloadable reports and figures

## Technical stack

```{list-table}
:header-rows: 1

* - Component
  - Technology
* - Backend
  - FastAPI, Python
* - Frontend
  - React, Chart.js, Leaflet
* - Data
  - Xarray, NetCDF4, PostGIS
* - Deployment
  - Docker, GitHub Actions
```

## Features

- Precipitation and temperature time series explorer
- NDVI anomaly browser
- Drought index calculator and visualizer
- Map-based spatial data viewer
- Report generator (PDF export)
