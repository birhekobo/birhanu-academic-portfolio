# Spatial Research Suite

## Overview

A comprehensive Python toolkit for satellite image processing and analysis, designed to streamline common remote sensing workflows.

## Objectives

- Provide a unified API for satellite data I/O, preprocessing, and analysis
- Implement state-of-the-art classification algorithms
- Enable reproducible research workflows
- Integrate with Google Earth Engine for cloud processing

## Methods

- **Classification** — random forest, SVM, CNN classifiers
- **Spectral indices** — NDVI, NDWI, MNDWI, NBR, SAVI, EVI
- **Change detection** — post-classification comparison, image differencing
- **Accuracy assessment** — confusion matrix, kappa, F1-score
- **Segmentation** — SLIC superpixels, multi-resolution segmentation
